## 🔄 Format technique pour les ajouts futurs

```
| Site | URL | Catégorie | Ville/Zone | Description courte | Compatible avec | Source | Date d'ajout |
```

**Champs** :
- **Catégorie** : 1-11 ci-dessus (ou demander à J pour catégorie 12+)
- **Compatible avec** : phase du menu social (Entrée Comprendre / Plat Vivre / Dessert Oser) + profils ciblés
- **Source** : `manuel J` / `crawler` / `partenaire` / `import depuis [autre source]`
- **Date d'ajout** : YYYY-MM-DD

### Règles d'intégrité (pour Claude Code)

1. **Aucun doublon** d'URL — vérifier avant ajout
2. **TOUJOURS visiter le site** (WebFetch ou WebSearch) avant catégorisation. **Ne jamais deviner depuis le nom de domaine** — c'est l'erreur classique. En cas d'impossibilité, demander à J.
3. **Vérifier ville et type réel** : un site `xxx.games` peut être un éditeur de jeux, un lieu physique, ou une marque produit — l'URL ne dit rien.
4. **Suivre l'arbre décisionnel `02_ARBRE_DECISIONNEL.md`** étape par étape. Pas de classement au feeling.
5. **Compter chaque ajout / suppression / modification** dans la section "Statistiques actuelles" en haut
6. **Datage** automatique à chaque modification dans une section "Historique" (V1+)
